# Protect the Child

A 2D side-scrolling action game built with Unity.
Fight off waves of enemies while protecting a vulnerable companion. Survive as long as you can as the difficulty increases over time.

## !Play the Game

https://phuonganh284.itch.io/protect-the-child

## !Controls

* Move: A / D
* Jump: Space
* Attack: Left Mouse Button

## !Features

* Player movement and combat system
* Enemy behavior (movement, detection, attack)
* Dynamic enemy spawning with increasing difficulty
* Health, damage, and game over system

## !Tech Stack

* Unity
* C#

## !Notes

This project was built as a learning project with additional custom features and improvements.
